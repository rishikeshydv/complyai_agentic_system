# Curated Regulatory Dataset

This repository ships an internal curated dataset for AML/BSA and sanctions citations.
No external web calls are used at runtime for citation retrieval.

Disclaimer:
- These citations are curated references intended for configurable compliance workflows.
- Legal teams must review and customize mappings for production use.

Also included:
- `mapping_v2.yaml`: deterministic law-mapping v2 seed (typologies, controls, control-to-citation mappings, rule maps).
